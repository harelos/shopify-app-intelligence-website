# Shopify App Intelligence

Landing page for the **Shopify App Intelligence** Chrome extension — an
AI-driven B2B tool that analyzes any Shopify store's installed apps and tech
stack for competitive research, lead scoring, and sales prospecting.

Static site, no build step: `index.html` + `privacy.html` + `support.html`.

## Pages

- `index.html` — landing page, pricing, feature breakdown
- `privacy.html` — privacy policy (Chrome Web Store compliant)
- `support.html` — support / FAQ

## Deploy

Push to GitHub Pages, or serve `index.html` from any static host.
